## 2325. Decode the Message (easy)

```python
class Solution:
    def decodeMessage(self, key: str, message: str) -> str:
        d = {}
        d[' '] = ' '
        index = 0
        for i, elem in enumerate(key):
            if elem == ' ': continue
            if elem in d: continue
            d[elem] = chr(index + ord('a'))
            index += 1

        # print(d)
        return ''.join([d[x] for x in message])
```
