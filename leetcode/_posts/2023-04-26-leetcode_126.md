## 258. Add Digits (easy)

```python
class Solution:
    def addDigits(self, num: int) -> int:
        res = 0
        if num < 10:
            return num
        while num > 0:
            print(num, res)
            res += num % 10
            num = num // 10
        return self.addDigits(res)
```
