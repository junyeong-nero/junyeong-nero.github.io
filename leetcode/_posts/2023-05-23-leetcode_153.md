## 703. Kth Largest Element in a Stream (easy)

```python
class KthLargest:

    def __init__(self, k: int, nums: List[int]):
        self.d = sorted(nums)
        self.k = k
        

    def add(self, val: int) -> int:
        index = bisect_right(self.d, val)
        self.d.insert(index, val)
        return self.d[-self.k]
```
