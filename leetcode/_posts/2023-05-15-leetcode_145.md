## 1721. Swapping Nodes in a Linked List (medium)

```python
class Solution:
    def swapNodes(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        arr = []
        temp = head
        while temp:
            arr.append(temp)
            temp = temp.next

        n = len(arr)
        a, b = arr[k - 1].val, arr[n - k].val
        # print(a, b)
        arr[k - 1].val = b
        arr[n - k].val = a

        return head
```
