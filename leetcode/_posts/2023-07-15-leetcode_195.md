## 848. Shifting Letters (medium)

```python
class Solution:
    def shiftingLetters(self, s: str, shifts: List[int]) -> str:

        def shift_char(c, num):
            num %= 26
            p = (ord(c) - ord('a') + num) % 26
            return chr(p + ord('a'))

        res = ''
        n = len(s)
        cur = 0
        for i in range(n - 1, -1, -1):
            c = s[i]
            cur += shifts[i]
            res += shift_char(c, cur)
        
        return res[::-1]
```
