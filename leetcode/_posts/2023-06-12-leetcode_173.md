## 228. Summary Ranges (easy)

```python
class Solution:
    def summaryRanges(self, nums: List[int]) -> List[str]:
        i = 0
        n = len(nums)
        res = list()
        while i < n:
            j = i
            while j + 1 < n and nums[j + 1] - nums[j] == 1:
                j += 1
            
            temp = str(nums[i])
            if j > i:
                temp += '->'
                temp += str(nums[j])
            res.append(temp)
            i = j + 1
        return res
```
