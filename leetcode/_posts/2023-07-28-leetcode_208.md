## 486. Predict the Winner (medium)

DFS Approach

```python
class Solution:
    def PredictTheWinner(self, nums: List[int]) -> bool:
        if len(nums) == 1:
            return True

        def dfs(i, j):
            if i > j:
                return math.inf
            if i == j:
                return nums[i]
            temp = max(nums[i] - dfs(i + 1, j), nums[j] - dfs(i, j - 1))
            return temp

        p = dfs(0, len(nums) - 1)
        print(p)
        return p >= 0
            

```
