## 2348. Number of Zero-Filled Subarrays (medium)

```python
class Solution:
    def zeroFilledSubarray(self, nums: List[int]) -> int:
        n = len(nums)
        i = 0
        res = 0
        while i < n:
            j = i
            while j < n and nums[j] == 0:
                j += 1
            p = j - i
            # print(p)
            res += p * (p + 1) // 2
            i = j + 1
            # [0, 0] => 3
            # m + m - 1 + ... 1 = m (m + 1) / 2
        return res

            
```
