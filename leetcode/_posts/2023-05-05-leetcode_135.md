## 1456. Maximum Number of Vowels in a Substring of Given Length (medium)

```python
class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        n = len(s)
        counter = Counter(s[:k])
        res = -1
        for i in range(k, n):
            res = max(res, counter['a'] + counter['e'] + counter['i'] + counter['o'] + counter['u'])
            a, b = s[i], s[i - k]
            counter[a] += 1
            counter[b] -= 1
        
        res = max(res, counter['a'] + counter['e'] + counter['i'] + counter['o'] + counter['u'])
        
        return res
```
