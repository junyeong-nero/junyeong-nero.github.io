## 1046. Last Stone Weight (easy)

```python
class Solution:
    def lastStoneWeight(self, stones: List[int]) -> int:
        heap = []
        for stone in stones:
            heappush(heap, -stone)

        while len(heap) >= 2:
            # print(heap)
            a, b = -heappop(heap), -heappop(heap)
            if a > b:
                heappush(heap, b - a)
        
        # print(heap)
        return -heap[0] if heap else 0
```
