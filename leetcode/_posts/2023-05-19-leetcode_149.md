## 785. Is Graph Bipartite? (medium)

```python
class Solution:
    def isBipartite(self, graph: List[List[int]]) -> bool:
        n = len(graph)
        visited = [0] * n
        def dfs(cur, flip):
            if visited[cur] != 0:
                return flip == visited[cur]
            visited[cur] = flip
            for node in graph[cur]:
                temp = dfs(node, -flip)
                if not temp:
                    return False
            return True

        for i in range(n):
            if visited[i] != 0:
                continue
            temp = dfs(i, 1)
            if not temp:
                return False

        return True
```
