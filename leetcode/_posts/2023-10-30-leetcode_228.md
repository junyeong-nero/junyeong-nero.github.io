## 1356. Sort Integers by The Number of 1 Bits

with bin.count

```python
class Solution:
    def sortByBits(self, arr: List[int]) -> List[int]:
        arr.sort()
        return sorted(arr, key=lambda x : bin(x).count('1'))
        
```
