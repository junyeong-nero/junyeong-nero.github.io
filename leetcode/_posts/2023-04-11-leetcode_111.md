## 2390. Removing Stars From a String (medium)

```python
class Solution:
    def removeStars(self, s: str) -> str:
        stack = [] 
        for char in s:
            if char == '*':
                stack.pop()
            else:
                stack.append(char)
        return ''.join(stack)
```
