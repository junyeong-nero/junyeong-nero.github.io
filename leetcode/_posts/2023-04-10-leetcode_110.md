## 20. Valid Parentheses (easy)

with Stack 

```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack = []
        open_ = '({['
        close_ = ']})'
        for p in s:
            if p in open_:
                stack.append(p)
            if p in close_:
                c = ''
                if stack:
                    c = stack.pop()
                if c == '[' and p == ']':
                    continue
                if c == '(' and p == ')':
                    continue
                if c == '{' and p == '}':
                    continue
                return False
        return len(stack) == 0
```
