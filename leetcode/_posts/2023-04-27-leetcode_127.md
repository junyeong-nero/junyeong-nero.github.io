## 319. Bulb Switcher (medium)

```python
class Solution:
    def bulbSwitch(self, n: int) -> int:
        return (int)(n ** 0.5)
```

## 672. Bulb Switcher II (medium)

```python
class Solution:
    def flipLights(self, n, p):
        return [[1,1,1], [2,3,4], [2,4,7], [2,4,8]][min(p, 3)][min(n - 1, 2)]
```
