## 1140. Stone Game II (medium)

DFS Approach

```python
class Solution:
    def stoneGameII(self, piles: List[int]) -> int:

        @cache
        def dfs(cur, M):
            prev = cur
            res = 0
            for X in range(1, 2 * M + 1):
                if cur >= len(piles):
                    break
                temp = dfs(cur + 1, max(M, X))
                res = max(res, sum(piles[prev:]) - temp)
                cur += 1

            return res

        return dfs(0, 1)
```
