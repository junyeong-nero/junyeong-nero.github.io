## 23. Merge k Sorted Lists (hard)

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def mergeKLists(self, lists: List[Optional[ListNode]]) -> Optional[ListNode]:
        q = []
        for head in lists:
            cur = head
            while cur:
                heappush(q, cur.val)
                cur = cur.next
            
        res = ListNode(0)
        cur = res
        while q:
            target = heappop(q)
            cur.next = ListNode(target)
            cur = cur.next

        return res.next
```
