## 2244. Minimum Rounds to Complete All Tasks (medium)

```python
class Solution:
    def minimumRounds(self, tasks: List[int]) -> int:
        counter = collections.Counter(tasks)
        res = 0
        for p in counter.values():
            # print(p)
            if p < 2:
                return -1
            else:
                res += math.ceil(p / 3)
                # divmod 를 이용하면 더 빨라지더라.
        return res     
```
