## 520. Detect Capital (easy)

```python
class Solution:
    def detectCapitalUse(self, word: str) -> bool:
        first = word[0].isupper()
        n = len(word)
        if first:
            for i in range(1, n - 1):
                a, b = word[i].isupper(), word[i + 1].isupper()
                if a != b:
                    return False
        else:
            for i in range(1, n):
                if word[i].isupper():
                    return False
        return True
        
```
