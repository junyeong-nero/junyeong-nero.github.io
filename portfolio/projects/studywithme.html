StudyWithMe

2021.01 - 2021.02

📋 Overview

A management application designed for small-scale educational academies. It facilitates the management of student data, attendance, and teacher scheduling.

🛠️ Technologies

Kotlin Android Firebase

⚡ Features

  • Student Management - Track student information and progress
  • Attendance Tracking - Easy check-in/check-out system
  • Teacher Scheduling - Manage class schedules
  • Firebase Backend - Real-time data synchronization
