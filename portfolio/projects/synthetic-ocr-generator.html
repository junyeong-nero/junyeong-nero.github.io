Synthetic OCR Image Generator

2024

ğŸ“‹ Project Overview

A comprehensive synthetic data generation tool for creating OCR (Optical Character Recognition) training datasets. The system generates realistic text images with diverse fonts, backgrounds, and augmentations, addressing the critical shortage of labeled Korean OCR training data. The generated dataset has been published on HuggingFace for community use.

ğŸ¯ Problem Definition & Goals

  • Problem: Training robust OCR models requires large labeled datasets, but creating real-world OCR data is expensive and time-consuming. Korean text recognition particularly suffers from limited training data availability.
  • Goal 1: Build a configurable text image generator that produces diverse, realistic training samples.
  • Goal 2: Support Korean text with proper font rendering and character coverage.
  • Goal 3: Implement augmentations that mimic real-world image degradation (blur, noise, perspective distortion).

âš™ï¸ Key Features & Contributions

  • Synthetic Text Generation: Creates text images with controllable parameters including font size, color, spacing, and alignment.
  • Multi-Font Support: Integrates various Korean and English fonts to increase training data diversity.
  • Background Variation: Generates diverse backgrounds including solid colors, gradients, textures, and real-world image crops.
  • Data Augmentation: Applies realistic transformations including Gaussian blur, noise injection, perspective warping, and JPEG compression artifacts.
  • Batch Generation: Efficient multiprocessing for large-scale dataset creation with progress tracking.

ğŸ”§ Technical Challenges & Solutions

  • Font Coverage: Many fonts lack complete Korean character support. Implemented font validation to ensure complete Hangul syllable coverage.
  • Realistic Rendering: Initial synthetic images looked too clean. Added antialiasing variations and subtle rendering imperfections.
  • Augmentation Balance: Over-augmented images became unreadable. Implemented parameterized augmentation intensities with reasonable defaults.
  • Memory Efficiency: Generating millions of images required optimization. Used lazy generation and efficient image processing pipelines.

ğŸ“ˆ Results & Learnings

  • Dataset Publication: Released Korean synthetic OCR dataset on HuggingFace with thousands of diverse training samples.
  • OCR Improvement: Models trained with synthetic data showed improved robustness to font variations and image degradation.
  • Community Adoption: Dataset available for download and use by OCR researchers working on Korean text recognition.
  • Key Learning: Gained expertise in synthetic data generation, image processing, and the art of creating training data that transfers well to real-world applications.
